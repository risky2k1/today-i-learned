## Programming

Hôm nay tìm hiểu về cách sử dụng `async/await` trong JavaScript để xử lý các tác vụ bất đồng bộ một cách mượt mà hơn. Thay vì dùng Promise chain, `async/await` giúp code trông synchronous nhưng vẫn non-blocking, dễ đọc và maintain hơn.

```javascript
async function fetchData() {
  const response = await fetch('https://api.example.com/data');
  const data = await response.json();
  return data;
}
```
