## Programming

*   **`str.startswith()` và `str.endswith()`:** Hai hàm này cực kỳ hữu ích để kiểm tra xem một chuỗi có bắt đầu hoặc kết thúc bằng một chuỗi con cụ thể hay không. Mình đã từng dùng `if string.startswith("http")` để lọc các URL.
*   **List Comprehensions:**  Cách viết ngắn gọn và hiệu quả để tạo ra một danh sách mới dựa trên một danh sách hiện có. Ví dụ: `[x * 2 for x in range(10)]` sẽ tạo ra một danh sách các số từ 0 đến 18.
*   **`f-strings` (Formatted String Literals):**  Cách dễ dàng nhất để chèn các biến vào chuỗi. Ví dụ: `name = "Lan"; print(f"Chào {name}!")`

**Nhận xét:** Mình thấy list comprehensions rất mạnh mẽ và sẽ cố gắng sử dụng chúng nhiều hơn trong các dự án sắp tới để code ngắn gọn và dễ đọc hơn.  Mình cũng cần luyện tập thêm để sử dụng `startswith()` và `endswith()` một cách hiệu quả hơn.

## Programming

Hôm nay mình tìm hiểu về cách sử dụng `async/await` trong Python để viết code bất đồng bộ (asynchronous) dễ đọc và dễ bảo trì hơn. Thay vì dùng callbacks lồng nhau, `async/await` giúp code trông giống như code đồng bộ, nhưng lại tận dụng tối đa tài nguyên khi chờ đợi các tác vụ I/O.  Việc này đặc biệt hữu ích khi làm việc với các API hoặc database.

*   `async` định nghĩa một hàm là coroutine.
*   `await` dùng để tạm dừng thực thi hàm cho đến khi một coroutine khác hoàn thành.

Mình sẽ thử áp dụng `async/await` vào project xử lý dữ liệu từ nhiều API để cải thiện hiệu suất.
