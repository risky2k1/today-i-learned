## DevOps

Học được cách dùng **multi-stage builds trong Docker** để tối ưu size image. Thay vì build và chạy trong cùng 1 stage, ta có thể tách riêng: stage 1 build source code với full dependencies, stage 2 copy artifacts cần thiết vào image runtime tối giản.

Ví dụ với Node.js:
```dockerfile
FROM node:18 AS builder
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

FROM node:18-alpine AS runtime
COPY --from=builder /app/dist ./dist
COPY --from=builder /app/node_modules ./node_modules
CMD ["node", "dist/index.js"]
```
Giảm được ~60% size image so với cách cũ.

- Kết hợp với `.dockerignore` để loại bỏ file không cần thiết.
