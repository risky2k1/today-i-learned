## Programming

Hôm nay tìm hiểu về **WebAssembly (WASM)** và cách nó giúp JavaScript chạy nhanh hơn. WASM cho phép code viết bằng các ngôn ngữ compiled (C/C++, Rust) chạy trực tiếp trong browser, giảm tải cho JS engine. Đặc biệt hữu ích cho các ứng dụng nặng về tính toán như games, image processing.

```javascript
// Ví dụ nhỏ: import một module WASM từ JS
import init, { add } from './calculator.js';

async function run() {
  await init();
  console.log(add(2, 3)); // 5
}
```

Sẽ thử build một demo nhỏ với Rust → WASM để cảm nhận performance khác biệt so với pure JS.
