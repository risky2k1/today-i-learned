## DevOps

Hôm nay tìm hiểu về Docker Compose - công cụ giúp định nghĩa và chạy multi-container Docker applications. Compose sử dụng file YAML để cấu hình services, networks và volumes, giúp triển khai môi trường development nhanh chóng và nhất quán giữa các máy.

- `docker-compose up` để start services, `docker-compose down` để stop và remove.
- Rất hữu ích cho việc setup local development với full-stack app (ví dụ: web app + database + cache).

Plan: Áp dụng Compose để standardize môi trường dev cho team, giảm thiểu "works on my machine" issues.
