## DevOps

Hôm nay tôi tìm hiểu về cách sử dụng Docker Compose để quản lý multi-container application. Docker Compose giúp định nghĩa và chạy nhiều container cùng lúc thông qua file `docker-compose.yml`, rất tiện lợi cho việc phát triển và test local.

Tôi đã thử setup một ứng dụng Node.js kết hợp với MySQL, và nhận ra việc config network và volume trong Compose file rất quan trọng để đảm bảo các service giao tiếp với nhau một cách ổn định.
