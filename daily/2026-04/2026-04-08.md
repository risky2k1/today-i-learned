Hôm nay tìm hiểu về cách sử dụng `kubectl` để debug pod bị lỗi. Lệnh `kubectl describe pod <tên-pod>` giúp xem chi tiết trạng thái và events, trong khi `kubectl logs <tên-pod> -f` hữu ích để theo dõi log real-time. Cảm thấy việc nắm rõ các lệnh debug này giúp giảm đáng kể thời gian troubleshoot khi production gặp sự cố. Plan là sẽ note lại checklist các bước debug vào repo chung để team dễ tra cứu.

## Programming

Hôm nay tìm hiểu về **WebAssembly (WASM)**, một công nghệ cho phép chạy code compiled từ các ngôn ngữ như C/C++/Rust trên trình duyệt với tốc độ gần như native. WASM giúp tối ưu performance cho các ứng dụng nặng như game, video editing, hoặc machine learning ngay trên web.

- WASM không thay thế JavaScript mà bổ trợ, thường dùng cho các module performance-critical.
- Các công cụ như Emscripten hoặc wasm-pack giúp biên dịch code C/Rust sang WASM dễ dàng.

Plan: Thử build một demo nhỏ với Rust + WASM để xem performance cải thiện thế nào so với JS thuần.
