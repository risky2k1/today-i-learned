## Programming

Hôm nay tìm hiểu về **Python's `__slots__`** - kỹ thuật giúp giảm bộ nhớ cho class bằng cách hạn chế thuộc tính động. Khi khai báo `__slots__`, Python sẽ không tạo `__dict__` cho mỗi instance, từ đó tiết kiệm khoảng 50% RAM so với class thông thường. Rất hữu ích khi làm việc với lượng lớn object như game engine hoặc data processing.

Hôm nay tìm hiểu về cách sử dụng Docker Compose để quản lý multi-container application. Docker Compose giúp định nghĩa và chạy nhiều container cùng lúc thông qua file `docker-compose.yml`, rất tiện cho việc phát triển và test môi trường.
