## DevOps

Hôm nay tôi tìm hiểu về **CI/CD pipeline với GitHub Actions**. Tôi đã cấu hình một workflow tự động chạy test và deploy khi push code lên branch `main`. Workflow sử dụng `ubuntu-latest` làm runner và chạy lệnh `npm test` trước khi deploy lên Heroku.
