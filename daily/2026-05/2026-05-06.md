## Programming

Hôm nay mình phát hiện memoization trong JavaScript có thể giúp tối ưu performance nhờ cache kết quả của function tính toán tốn thời gian. Thay vì tính lại từ đầu, function sẽ trả về kết quả đã lưu trong cache khi có cùng tham số. Mình đã áp dụng cho component React dùng hàm filter dữ liệu lớn và thấy tốc độ cải thiện đáng kể. Chúng ta nên xem xét memoization khi làm việc với các phép tính phức tạp hay danh sách dữ liệu lớn.

*Mình đang nghĩ đến việc áp dụng cho API call caching trong project hiện tại.*
