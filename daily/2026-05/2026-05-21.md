## Programming

Dùng `Array.prototype.at(-1)` để lấy phần tử cuối cùng của mảng thay vì `arr[arr.length - 1]` - vừa ngắn gọn, vừa đọc hơn.

```javascript
const arr = [1, 2, 3];
arr.at(-1); // 3
```

Đặc biệt hữu ích khi làm việc với stack/queue hoặc cần lấy last item trong chain of operations.
