## DevOps

- Multi-stage build giúp chia bước build và runtime thành các giai đoạn riêng, chỉ copy những gì cần thiết sang stage cuối.
- Ví dụ: build React app trong `builder` stage với node, chỉ copy thư mục `build` sang `nginx` stage duy nhất ~10MB.
- Mình vừa refactor image Docker giảm từ 450MB xuống còn 12MB — deploy nhanh hơn 3s trên CI/CD.

Plan: Tối ưu image Python service tiếp theo để giảm build time.
