## Programming

Hôm nay mình tìm hiểu về cách sử dụng `async/await` trong Python để viết code bất đồng bộ (asynchronous) dễ đọc và quản lý hơn. Thay vì dùng callbacks lồng nhau, `async/await` giúp code trông giống như code đồng bộ, nhưng lại tận dụng tối đa tài nguyên khi chờ đợi các tác vụ I/O.  Điều này đặc biệt hữu ích khi làm việc với các API hoặc database.

*   `async` định nghĩa một hàm là coroutine.
*   `await` dùng để tạm dừng thực thi hàm cho đến khi một coroutine khác hoàn thành.

Mình sẽ thử áp dụng `async/await` vào project xử lý dữ liệu từ nhiều API để cải thiện hiệu suất.
