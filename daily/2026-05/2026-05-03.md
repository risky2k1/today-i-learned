## Programming

Đêm qua mình discover được `useMemo` và `useCallback` trong React không chỉ dùng để tối ưu performance mà còn quan trọng để tránh object/function tạo mới mỗi lần render, gây re-render không cần thiết ở child component dùng `React.memo`.

- `useMemo`: cache kết quả tính toán, chỉ tính lại khi dependencies thay đổi
- `useCallback`: giữ nguyên reference function, phù hợp truyền prop cho child component
- Lưu ý dùng ESLint rule `react-hooks/exhaustive-deps` để tránh bug do dependencies thiếu

Mình thấy vô ích khi mới học vì sợ complicated, nhưng dùng thử trong project thật thì thật sự "đỡ" được nhiều re-render không cần thiết, đặc biệt với danh sách lớn hoặc component con phức tạp.

## DevOps

Hôm nay mình tìm hiểu về **GitOps** - một cách tiếp cận để quản lý infrastructure và application deployment bằng Git như single source of truth. Thay vì chạy manual commands, mọi thay đổi đều được commit vào Git repo, CI/CD sẽ tự động sync trạng thái thực tế với desired state.

**ArgoCD** và **Flux** là hai tool phổ biến để implement GitOps trên Kubernetes. Mình đang thử setup ArgoCD cho project hiện tại, hy vọng sẽ giảm được việc "deploy bằng tay" và dễ rollback hơn.
