## Programming

Mới phát hiện `Array.prototype.at()` trong JavaScript cho phép truy cập phần tử mảng từ cuối bằng chỉ số âm, giống như Python nhưng không cần slice.

```javascript
const arr = [1, 2, 3, 4, 5];
arr.at(-1); // 5
arr.at(-2); // 4
```

Rất hữu ích khi cần lấy phần tử cuối mà không muốn viết `arr[arr.length - 1]`. Sẽ áp dụng từ nay thay cho cách cũ.

## Programming  
Hôm nay mình phát hiện rằng Python hỗ trợ context manager (`with` statement) giúp quản lý tài nguyên tự động (như file, bộ nhớ) mà không cần gọi `.close()` thủ công. Cơ chế này dựa vào 2 phương thức `__enter__` và `__exit__` để thiết lập và dọn dẹp tài nguyên. Áp dụng để xử lý file lớn hoặc kết nối database giúp tránh rò rỉ bộ nhớ.  
*Nhận xét: Đúng là "explicit is better than implicit" nhưng context manager là cách làm cho explicit trở nên implicit mà sạch sẽ!*
