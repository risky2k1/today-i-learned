## Automation

Hôm nay mình tìm hiểu thêm về **GitHub Actions** và phát hiện có thể dùng `workflow_dispatch` để trigger workflow thủ công từ UI, rất tiện cho việc chạy deploy hay cleanup tasks mà không cần push code mới.

Bonus: Có thể truyền input parameters khi trigger thủ công, giúp linh hoạt hơn cho từng use case.

```yaml
on:
  workflow_dispatch:
    inputs:
      environment:
        description: 'Environment to deploy'
        required: true
        default: 'staging'
```
