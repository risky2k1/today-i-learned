## Travel Tech

Hôm nay mình tìm hiểu về **Giata API** - một dịch vụ cung cấp dữ liệu khách sạn (hình ảnh, tiện nghi, location) được nhiều OTA lớn như Booking.com sử dụng. Giata có hơn 2.5 triệu khách sạn trên toàn cầu, data được cập nhật thường xuyên từ các đối tác trực tiếp.

- Ưu điểm: RESTful API, documentation rõ ràng, hỗ trợ nhiều ngôn ngữ
- Phù hợp cho các dự án du lịch, so sánh giá khách sạn, hoặc xây dựng meta-search engine

**Plan:** Thử integrate Giata vào side project về so sánh giá phòng nội địa Việt Nam xem sao.



## Programming

- Vừa học được về async/await trong JavaScript: thay vì dùng promise chaining, giờ code dài hơn ít nhưng dễ đọc hơn nhiều.

Thấy async/await thực sự làm code trôi chảy hơn, nhất là khi làm việc với các flow phức tạp. Plan refactor ngay trên dự án này để cleaner code hơn!
