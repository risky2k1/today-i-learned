## Automation

Hôm nay mình đã khám phá cách sử dụng **GitHub Actions** để tự động triển khai một container Docker lên AWS ECS sau mỗi lần merge vào branch `main`.  
- Sử dụng `actions/checkout` để lấy mã nguồn, sau đó build image với `docker/build-push-action`.  
- Dùng `aws-actions/configure-aws-credentials` để thiết lập quyền truy cập AWS, cuối cùng chạy `aws ecs update-service` để cập nhật task definition.

Cảm giác thú vị khi thấy pipeline chạy hoàn toàn tự động, mình dự định sẽ mở rộng thêm các bước test tích hợp trước khi deploy.


## Programming  
Hôm nay mình học về `Optional Chaining` (toán tử `?.`) trong JavaScript để xử lý an toàn các thuộc tính lồng sâu. Ví dụ: `user?.profile?.address` sẽ trả về `undefined` nếu `user` hoặc `profile` là `null`/`undefined` thay vì gây lỗi.  
Mình sẽ áp dụng ngay vào code API response handling để tránh lỗi khi dữ liệu không như kỳ vọng. Thực tế thấy rõ sự khác biệt về độ an toàn so với cách kiểm tra điều kiện thủ công.
