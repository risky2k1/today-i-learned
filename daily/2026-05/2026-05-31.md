## Programming

- Hôm nay mình mới biết trong Python 3.12, có thể unpack dictionary trực tiếp trong `match` statement bằng cách dùng `case {"key": value, **rest}`:
  ```python
  match data:
      case {"name": name, "age": age, **extra}:
          print(f"{name} - {age} tuổi")
  ```
- Tính năng này giúp xử lý JSON hoặc dict phức tạp gọn gàng hơn, không cần phải access từng key một.
- Mình thử áp dụng ngay vào project xử lý API response, code ngắn hơn khá nhiều so với các cách trước.
