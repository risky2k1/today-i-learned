## DevOps

- Khi triển khai Docker Swarm, tôi đã học cách tận dụng **service update --image** để quay trở lại phiên bản cũ nếu curl check fail, tránh downtime kéo dài.  
- Dùng **docker inspect --format='{{.HostConfig.NetworkMode}}'** giúp nhanh xác định mạng đang dùng, thuận tiện khi debug issue mạng lưới container.  
- **Planning**: sẽ thử triển khai CI/CD với GitHub Actions + Docker Swarm để kiểm tra tự động rollback khi build lỗi.
