## Programming

*   **`str.startswith()` và `str.endswith()`:** Hai hàm này cực kỳ hữu ích để kiểm tra xem một chuỗi có bắt đầu hoặc kết thúc bằng một chuỗi con cụ thể hay không. Mình đã từng dùng `if string.startswith("http")` để lọc các URL.
*   **List Comprehensions:**  Cách viết ngắn gọn và hiệu quả để tạo ra một danh sách mới dựa trên một danh sách hiện có. Ví dụ: `[x * 2 for x in range(10)]` sẽ tạo ra một danh sách các số từ 0 đến 18.
*   **`f-strings` (Formatted String Literals):**  Cách dễ dàng nhất để chèn các biến vào chuỗi. Ví dụ: `name = "Lan"; print(f"Chào {name}!")`

**Nhận xét:** Mình thấy list comprehensions rất mạnh mẽ và sẽ cố gắng sử dụng chúng nhiều hơn trong các dự án sắp tới để code ngắn gọn và dễ đọc hơn.  Mình cũng cần luyện tập thêm về việc sử dụng `startswith()` và `endswith()` để tránh viết các đoạn code phức tạp hơn cần thiết.

## DevOps

Hôm nay mình phát hiện ra **uv** - package manager của Astral (nhà phát triển Ruff) - có tốc độ cài đặt pip nhanh gấp 10-100 lần so với pip truyền thống, đặc biệt hữu ích khi làm việc với Docker image.

```bash
# Cài đặt nhanh
curl -LsSf https://astral.sh/uv/install.sh | sh
uv pip install -r requirements.txt
```

- Dùng được cho cả Python 3.8+ và PyPy
- Tích hợp tốt với Docker multi-stage build
