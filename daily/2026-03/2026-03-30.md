## DevOps

Hôm nay tìm hiểu về cách dùng Docker Compose để quản lý multi-container application. Docker Compose giúp định nghĩa và chạy nhiều container cùng lúc thông qua file YAML, rất tiện lợi cho việc phát triển và testing.

- Sử dụng `docker-compose up` để khởi động toàn bộ services được định nghĩa trong file `docker-compose.yml`.
- Lệnh `docker-compose down` giúp dừng và xóa các container, network, volume đã tạo.
