## DevOps

Hôm nay tìm hiểu về cách cấu hình **Git hooks** để tự động chạy test trước khi push code. Git hooks là các script chạy tự động khi có sự kiện Git xảy ra, giúp phát hiện lỗi sớm và duy trì chất lượng code.

- Hook `pre-commit` có thể chạy ESLint và unit test.
- Hook `pre-push` thường dùng để chạy test toàn bộ trước khi push.

Plan: sẽ áp dụng pre-commit để lint code và chạy test tự động, giảm lỗi merge.
