## Programming  

Hôm nay tìm hiểu về **TypeScript utility types** - cụ thể là `Partial<T>` và `Pick<T>`. Hai utility này giúp làm việc với object types dễ dàng hơn khi cần chỉnh sửa hoặc lấy một phần properties. Ví dụ, `Partial<T>` chuyển tất cả properties thành optional, còn `Pick<T, K>` cho phép lấy ra một tập properties nhất định.

Cảm thấy đây là cách hay để tránh viết lại type nhiều lần, giúp code ngắn gọn và dễ maintain hơn. Sẽ áp dụng vào dự án React hook custom để handle form state.
