## DevOps

Hôm nay tìm hiểu về cách sử dụng GitHub Actions để tự động deploy static website lên GitHub Pages. Chỉ cần tạo file workflow `.github/workflows/deploy.yml` và config để trigger mỗi khi push lên branch `main`. Rất tiện lợi và không cần server riêng.

## Automation

Học được cách dùng Python `requests` + `BeautifulSoup` để crawl dữ liệu từ web đơn giản. Ví dụ: lấy danh sách tên các trường đại học ở Hà Nội từ Wikipedia. Kế hoạch sẽ thử áp dụng để tự động cập nhật dữ liệu cho project cá nhân.

## Travel Tech

Khám phá ứng dụng Maps.me - bản đồ offline hữu ích khi đi trekking ở vùng sâu vùng xa. Có thể tải trước map khu vực cần đi, đánh dấu waypoints và theo dõi lộ trình GPS. Dự định dùng cho chuyến đi Fansipan sắp tới.

## Programming

Hôm nay gặp lỗi `ModuleNotFoundError` khi chạy script Python. Nguyên nhân là do chưa activate virtual environment. Bài học: luôn nhớ activate venv trước khi chạy project để tránh xung đột dependencies.

## Trekking - Hiking - Camping (Các địa điểm ở Việt Nam)

Đọc review về trekking đỉnh Bà Đen (Tây Ninh). Đường lên khá dài (~1500m), có thể đi cáp treo hoặc trekking bộ. Nhiều người chọn đi bộ để ngắm cảnh và rèn sức khỏe. Plan: cuối tháng sẽ thử chinh phục đỉnh này cùng nhóm bạn.
