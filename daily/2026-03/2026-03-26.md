## DevOps

Hôm nay tìm hiểu về **Docker Compose** - cách định nghĩa và chạy multi-container applications. Compose sử dụng YAML file để cấu hình services, networks và volumes, giúp đơn giản hóa việc quản lý các containers liên quan. Ví dụ: `docker-compose up -d` sẽ start toàn bộ stack chỉ với một lệnh.

Điểm hay là có thể version control file compose, giúp team dễ chia sẻ môi trường dev giống nhau. Plan: sẽ áp dụng cho dự án NodeJS + MongoDB đang làm.
