# 2026-06-30

## DevOps

Hôm nay học được cách triển khai CI/CD hiệu suất cao bằng Jenkins, tích hợp chi tiết các giai đoạn build, test tự động và deploy multi-environment. Một điểm nổi bật là việc dùng environment variables để quản lý cấu hình quy mô khác nhau, giúp tăng tính linh hoạt trong workflow. Các plugin thiết kế giúp trực quan hóa pipeline từ source code đến production server.  
Lý do quan trọng nhất để áp dụng: Hiệu chỉnh quy trình triển khai giúp giảm thời gian testing đến 50% khi so sánh với việc setup thủ công, đồng thời tối ưu hóa biến số cấu hình giúp dễ scale project cao hơn.
