## Programming  
Hôm nay mình phát hiện `contextlib.suppress` trong Python – một cách thay thế gọn gàng cho `try/except` khi muốn bỏ qua lỗi kiểu `FileNotFoundError` hoặc `KeyError`. Thay vì dùng `try: ... except: pass`, mình có thể viết `with contextlib.suppress(FileNotFoundError):` để code sạch hơn hẳn.  
Mình sẽ áp dụng cách này vào dự án xử lý file để giảm thiểu boilerplate code.



## Programming

- Mình vừa tìm hiểu về **f-strings** trong Python - thay vì dùng `+` để nối chuỗi, giờ dùng f-strings là code ngắn gọn hơn và ít lỗi syntax.
- Ví dụ: `f"Xin chào {tên}, bạn {tuổi} tuổi"` thay vì `"Xin chào " + tên + ", bạn " + str(tuổi) + " tuổi"`.
- Thực tế thì f-strings còn hỗ trợ format số, canh lề ngay trong chuỗi luôn, rất tiện cho việc in bảng kết quả.
- **Plan**: Áp dụng f-strings cho tất cả project nhỏ trong tuần tới để quen dần.
