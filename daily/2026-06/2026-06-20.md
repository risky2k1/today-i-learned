## Programming

Hôm nay mình mới tìm hiểu được về `useMemo` và `useCallback` trong React - hai hook thường dùng để tối ưu performance. `useMemo` giúp cache kết quả tính toán, chỉ tính lại khi dependencies thay đổi, còn `useCallback` dùng để giữ nguyên function reference giữa các lần render.

- Dùng `useMemo` cho các phép tính tốn CPU hoặc tạo object/array phức tạp  
- Dùng `useCallback` khi truyền function xuống component con dùng như dependency
- Nếu không có dependencies thay đổi, hai hook này giúp giảm số lần component con re-render không cần thiết

Ủy, áp dụng vào dự án nextjs đang làm thì component con sẽ nhanh hơn nhiều khi nhận về props function không đổi ✅.
