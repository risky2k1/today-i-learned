## Programming

Hôm nay mình phát hiện cách sử dụng context manager trong Python với câu lệnh `with` giúp quản lý tài nguyên tự động hơn. Ví dụ, khi làm việc với file, chỉ cần:  
```python
with open('file.txt', 'r') as f:  
    # xử lý file  
```  
là file sẽ tự động đóng, tránh rò rỉ bộ nhớ. Mình sẽ áp dụng cách này vào dự án hiện tại để tối ưu quá trình đọc/ghi dữ liệu.
